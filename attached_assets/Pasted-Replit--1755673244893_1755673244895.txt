Replit 웹사이트 생성용 마스터 프롬프트 (유니컵컴퍼니)

역할
당신은 “프랜차이즈 본사/식음료 테크 법인”의 코퍼레이트 웹사이트를 제작하는 시니어 프론트엔드 개발자이자 아트디렉터입니다. 결과물은 정적 배포가 가능한 React(Vite) + Tailwind CSS 기반 단일 리포지토리로 완성합니다. 코드는 깔끔한 컴포넌트 구조, 접근성(ARIA), SEO/OG 메타, 모바일 퍼스트를 기본으로 합니다.

1) 프로젝트 설정

Stack: React + Vite + Tailwind CSS + TypeScript

폴더:

/public/images (아래 자산 업로드)

/src/components, /src/sections, /src/styles

라우팅: 랜딩(홈) /, 소개 /about, 사업영역 /business, PR/뉴스 /news, 문의 /contact

글꼴: Pretendard(한글 본문), Inter(영문 UI). Google Fonts 혹은 CDN 사용.

컬러(브랜드 블루 기반):

--uc-blue: #1F3B83

--uc-deep: #0D1220

--uc-gray: #8A93A6

--uc-bg: #0B0E14 (히어로/푸터 다크톤)

--uc-accent: #29A3FF (하이라이트 라인)

다크한 프리미엄 톤, 여백 넓게, 라운드 14–16px, 카드 섀도우 절제.

3) 정보구조(IA) & 섹션 상세

CTA 2개: 가맹/제휴 문의, 회사 소개서(PDF) 다운로드(더미 링크)

우측에 logo-uc-company.png를 미묘한 모션 페이드인.

로고 2개 좌우 배치: logo-uc-company.png, logo-unicup-coffee.png

4) 공통 컴포넌트 & 품질 기준

<Header> 스크롤 고정, active 링크 표시.

<PrimaryButton> / <GhostButton> 재사용.

애니메이션: Framer Motion의 가벼운 페이드/슬라이드만.

반응형: 360/768/1024/1440 브레이크포인트.

접근성: 키보드 포커스 스타일, 명도대비 4.5:1 이상.

SEO: title, meta description, og:image=/images/slogan.jpg, lang="ko", 스키마 Organization.

5) 카피(초안 텍스트 – 코드에 포함)

헤드라인: “유니컵컴퍼니 – 합리적인 프리미엄을 표준화하다”

서브: “매뉴얼·품질·공급망을 한 화면에서 설계하는 프랜차이즈 본사”

CTA 버튼: “가맹/제휴 문의하기”

6) 문의 폼(스팸 대비)

필드: 성함/회사·점포/연락처/이메일/관심분야(가맹·제휴·공급)/메시지

Replit 환경변수로 Formspree 등 엔드포인트 분리. 제출 시 토스트 알림.

7) 코드 산출 요구

실행 가능한 전체 코드 생성: index.html 대신 Vite 구조.

tailwind.config.js, postcss.config.js, src/main.tsx, src/App.tsx 등 포함.

모든 섹션은 독립 파일(src/sections/Hero.tsx 등).

이미지 경로는 /images/... 정적 참조.

README에 배포 방법(Replit/Netlify), 이미지 교체 가이드 기재.
